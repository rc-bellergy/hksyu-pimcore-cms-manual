# Staffs

## Create a new staff

Go to `Data Objects` > `Your directory` > `Staff` , then right click ` Add Object` > `Staff`

![](images/staff01.png)

Please read and follow the Guide below, then fill up the form.

![](images/staff02.png)

* **Priority**  
The order numbers from greatest to least.
